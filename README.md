# Residency-Bot
A Discord bot that tracks the online residency hours of TLS staffers.

# Run locally
1. Run `npm install discord.js`
2. Create a `config.json` file with the following content
```
{
	"BOT_TOKEN": "<>",
	"APP_ID": "<>",
	"PUBLIC_KEY": "<>",
	"AUTH": "<>",
	"CLIENT_ID": "<>"
}
```
3. Start with `node index.js`
